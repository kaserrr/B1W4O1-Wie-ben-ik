Wie ben ik.

Waarom heb ik voor de opleiding gekozen?

Ik heb voor deze opleiding gekozen omdat ik het idee van zelf software of games maken echt super tof vind. En daardoor ben ik erg gemotiveerd om te leren programmeren.

Hoe zorg ik ervoor dat ik gemotiveerd blijf?

Ik blijf gemotiveerd omdat ik de opdrachten tot nu toe erg leuk vind om te doen. En daardoor krijg ik meer zin om eraan te werken.

Wat ben ik bereid om na de lessen nog te doen aan de opleiding?

Ik ben bereid om na te lessen ook nog een langer door te werken aan de opdrachten en dat geldt ook voor de weekenden. Want ik loop achter omdat ik later kwam instromen op de opleiding.

Waar wil ik mijzelf in ontwikkelen?

Ik wil leren om zelfstandig te kunnen programmeren.

Welke vooropleidingen heb ik gedaan en hoe kan ik de kennis daarvan toepassen op deze opleiding?

Ik heb hiervoor 1 jaar op mediavormgeving gezeten en ik zou naast het programmeren mijn projecten ook kunnen vormgeven.

Welke bij baantjes heb ik gehad en hoe zou ik de kennis daarvan binnen de opleiding kunnen toepassen?

Mijn eerste en enige bijbaan die ik heb gehad is bij een restaurant als keukenhulp. Ik denk niet dat ik iets van die bijbaan kan meenemen in de opleiding. Ik denk dat ik als vaardigheid heb geleerd om makkelijk met nieuwe mensen te praten die ik nog niet ken.
